# VoxForge — Roadmap

> Planned features and enhancements, ordered by priority

---

## ✅ Shipped (v1.0)

- [x] Voice Cloning from audio samples
- [x] Text-to-Speech with emotion, speed, pitch control
- [x] Voice Design from natural language descriptions
- [x] Voice Library with search, preview, delete
- [x] Multi-model architecture (8 engine plugins)
- [x] SSE-based model loading with progress streaming
- [x] Sound Effects / Foley generation (Bark)
- [x] Cross-lingual voice dubbing (CosyVoice, XTTS)
- [x] Podcast auto-generation (F5-TTS, Fish Speech)
- [x] Audio Inpainting
- [x] Dark glassmorphism design system
- [x] Server logs viewer
- [x] pnpm monorepo structure

---

## 🔜 Next Up (v1.1)

- [ ] **Voice Morphing** — Blend two voice embeddings mathematically
- [ ] **Gender / Pitch Swapping** — Mathematical pitch transformation
- [ ] **Browser Microphone Recording** — Record voice samples directly in-app
- [ ] **Voice Export / Import** — Share voice files between devices
- [ ] **Error Boundaries** — Graceful frontend error handling

---

## 🔮 Future (v2.0)

- [ ] **Real-time Streaming Generation** — WebSocket-based audio streaming
- [ ] **SSML Support** — Fine-grained speech markup
- [ ] **Batch TTS Generation** — Queue multiple texts for generation
- [ ] **Multi-speaker Dialogue** — Assign voices to characters in a script
- [ ] **Voice Marketplace** — Share and discover community voices
- [ ] **Docker Deployment** — One-command setup with `docker compose`
- [ ] **API SDK / CLI** — Programmatic access for developers

---

## 🧪 Experimental / Exploring

- [ ] **Emotion Transfer** — Transfer emotion from a reference clip
- [ ] **Singing Voice Synthesis** — TTS models that can sing
- [ ] **Audio-to-Audio** — Style transfer between audio clips
- [ ] **Fine-tuning UI** — Train models on your own dataset
- [ ] **Mobile App** — React Native companion app
